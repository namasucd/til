# ワークスペースのルートディレクトリにフォルダ作成 → フォルダの中にmdファイル作成 → mdファイルをプレビュー
## ワークスペースのルートディレクトリにフォルダ作成
1. `ctrl + 0` でサイドビューに移動
1. `↑↓`でディレクトリを選択
    - ルートディレクトリには`tab`で移動
1. `ctrl + n → n`でフォルダを作成
    - デフォルトではキーバインドが存在しないので、割り当てる)
## フォルダの中にmdファイル作成
1. ディレクトリを選択した状態で`ctrl + n` 
1. 拡張子を含むファイル名を入力し`Enter`
## mdファイルをプレビュー
1. `ctrl + k → v` 
    - (参考：`ctrl + k → d` で差分表示)
